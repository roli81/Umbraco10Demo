# Sss.Umb9.Mutobo

Umbraco 9 based Magic Umbraco Toolbox.


1. Install .NET 6
    - https://dotnet.microsoft.com/en-us/download/dotnet/6.0
2. Clone the Repo
3. open shell 
4. cd ./Nuun.Demo
5. dot net run
5. go to the url showed in terminal and perform the umbraco installtion
6. perform a uSync-Import (full)
